About Me

I am a former case worker who loves to create apps with code.

I first tried my hands out with coding in 2020 during a period when everyone was stuck indoors. A friend who works within the tech industry told me about JavaScript and all the cool things you can build with it. I was intrigued and jumped straight into the world of web development. Before I knew it, I was hooked.

Whilst managing my full-time job, I worked on numerous projects ranging from a tip calculator to a memory game. I thoroughly enjoyed the process of building innovative apps, and debugging errors in my code. This passion propelled me to learn a range of technologies with the aim of creating solutions to difficult problems.

After almost two years of self-teaching, I decided this was what I want to do for my career. During this time, I amassed a portfolio of personal projects that demonstrate my skill and expertise as a web developer. My projects focus on the client side, comprising of front-end technologies such as React. However, I am currently spending my evenings learning back-end technologies such as Node.JS, Express and MongoDB. My long term goal is to become an effective full stack developer, able to efficiently turn great ideas into working products.

My Work

Check out my projects below

Movie Watchlist App

A React application that fetches data from a movie database API. It dynamically renders information on most movies and tv series. Additionally, you are able to add a movie/tv series to a watchlist, which is saved to local storage.


Built with: React, Context API, Router, TMDB API, Material UI, Daisy UI

CV Maker

A react application which takes users data inputs, and prints a CV using the entered data. This project uses controlled forms, hooks and allowed me to practice using the context API state management. The CV printing feature was built with JavaScript.


Built with: React, Context API, CSS

Brochure Website

A website built for a local posting & printing company, which I have overseen over the course of a year. I took the approach of using a CSS framework, Bootstrap, to create a professional website design, and vanilla JavaScript for the functionality.


Built with: HTML, Sass, JavaScript

Note Keeper

A simple React CRUD application which allows users to create and store notes for display. As is the case with CRUD apps, they are also able to read, update and delete notes. This project was an opportunity to practice React Hooks.


Built with: React, CSS
